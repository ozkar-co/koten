# IA de lexico por raices

- Proposito: generar palabras nuevas a partir de raices conceptuales.
- Enfoque inicial: modelo pequeno y local, orientado a consistencia semantica.
- Uso esperado: sugerencias de vocabulario, traduccion interna y busqueda por concepto.
- Restriccion KISS: comenzar con heuristicas simples antes de modelos complejos.
