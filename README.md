# Koten

Koten ("el lugar donde nace la vida") es el backend de un mundo de rol personal.

El proyecto organiza y conecta el conocimiento vivo del mundo: historias, razas, idiomas, religiones, recursos de juego, hojas de personaje, habilidades, objetos, reglas y descripciones de elementos.

Tambien incluye la base para un sistema de generacion de palabras a partir de raices conceptuales, para apoyar traducciones y busquedas dentro de los idiomas del mundo.

Este repositorio esta pensado para uso local y personal, con un enfoque KISS: pequeno, claro, eficiente y facil de mantener.
