# Rasgos y Niveles

## Escala de Competencia

Cada rasgo tiene un nivel que indica tu maestría:

| Nivel | Nombre | Tirada |
|-------|--------|--------|
| 0 | Ignorante | 0 dados |
| 1 | Mediocre | 1 dado |
| 2 | Competente | 2 dados |
| 3 | Experto | 3 dados |
| 4 | Maestro | 4 dados |
| 5 | Leyenda | 5 dados |

## Rasgos Iniciales

En creación, los rasgos se comienzan en nivel 1, 2 o 3 según cómo distribuyas los 12 puntos.

## Aumento de Rasgos

Los rasgos pueden mejorar de los siguientes modos:

1. **Gastando 3 Puntos de Destino** - Subir un rasgo un nivel
2. **Creando un Rasgo Nuevo** - 3 Puntos de Destino para crear uno desde cero
3. **En el Tiempo de Juego** - El Master puede otorgar mejoras si el juego ha avanzado significativamente

## Penalización por Consecuencias

Las consecuencias penalizan tus rasgos:

- Una consecuencia menor penaliza un rasgo -1 durante X turno/sesión
- Una consecuencia mayor penaliza un rasgo -1 indefinidamente hasta recuperarse
- Si **todos tus rasgos llegan a 0** por penalizaciones, mueres o quedas incapacitado
