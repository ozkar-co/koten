# Reglas

Sistema de rol basado en **xD6 + Fate Acelerado**.

## Principios

- La creación de historia es el objetivo principal
- Todos los elementos (personajes, objetos, enemigos, lugares) pueden tener trasfondo, rasgos, estrés y consecuencias
- El éxito se mide por diversión y consenso
- La defensa es automática
- La rendición es siempre una opción negociable
